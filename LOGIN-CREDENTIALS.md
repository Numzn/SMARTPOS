## 🚀 Quick Start Instructions

1. **Start Both Servers:**
   - Double-click `start-smartpos.bat` to start both backend and frontend
   - OR start them manually in separate terminals:

2. **Manual Backend Start:**
   ```bash
   cd C:\Users\NUMERI\POSPROJECT\smart-pos-backend
   npm run dev
   ```

3. **Manual Frontend Start:**
   ```bash
   cd C:\Users\NUMERI\POSPROJECT\smart-pos-frontend
   npm run dev
   ```

4. **Access the Application:**
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:3000
